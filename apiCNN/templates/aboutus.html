Proyecto Interciclo | Convolutional Neuronal Network

En este proyecto se implementa redes neuronales para la clasificación de imágenes, para lo cual se utilizó como base de entrenamiento y pruebas un repositorio publicado por Intel.

Dicho repositorio presenta imágenes con las siguiente etiquetas:

  •   Montaña
  •   Calle
  •   Edificios
  •   Bosque
  •   Mar
  •   Glaciar
